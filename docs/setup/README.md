# 🛠️ 환경 설정 및 도구 사용법

이 폴더는 **개발 환경 설정 및 도구 사용법**을 포함합니다.

## 📁 포함 문서

### window 환경에서 클로드 코드 및 관련 설치.txt
- **목적**: Windows 환경에서 Claude Code 설치 가이드
- **내용**:
  - Claude Code 설치 방법
  - 필수 도구 설치 (Node.js, Java, Docker 등)
  - 환경 변수 설정
  - 초기 설정
- **대상**: Windows 사용자
- **용도**:
  - 최초 개발 환경 구성
  - 새로운 PC에서 환경 설정

### 클로드 코드 단축키 및 사용법.txt
- **목적**: Claude Code 효율적 사용법
- **내용**:
  - 주요 단축키
  - 유용한 명령어
  - 생산성 팁
  - 트러블슈팅
- **대상**: Claude Code 사용자
- **용도**:
  - 빠른 참고 자료
  - 생산성 향상

## 🎯 사용 방법

### 최초 설정
1. `window 환경에서 클로드 코드 및 관련 설치.txt` 참고하여 환경 구성
2. 모든 도구 설치 완료 후 프로젝트 클론

### 일상적 사용
- `클로드 코드 단축키 및 사용법.txt`를 참고하여 효율적으로 작업
- 자주 사용하는 명령어는 따로 메모

## 📝 파일 형식

**왜 .txt 파일인가?**
- 간단한 참고 자료는 Markdown보다 텍스트 파일이 더 가볍고 빠름
- 어떤 에디터에서도 바로 열람 가능
- 설정 값 복사-붙여넣기 용이
